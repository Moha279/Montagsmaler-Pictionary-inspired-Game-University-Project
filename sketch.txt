MVC-Pinciple:

┌────────────┐
│   View     │  ← Zeichenfläche (JavaFX)
│            │  → zeigt das Bild, Buttons, Ergebnis
└────┬───────┘
     │ z.B. Mausbewegung, Klick
     ▼
┌────┴───────┐
│ Controller │  ← steuert den Ablauf
│            │  → verarbeitet Eingaben
└────┬───────┘
     │ z.B. sagt dem Model: „Hier wurde gezeichnet!“
     ▼
┌────┴───────┐
│   Model    │  ← speichert das Bild, wertet mit KI aus
│            │  → gibt Ergebnis (z. B. 89 %)
└────────────┘
